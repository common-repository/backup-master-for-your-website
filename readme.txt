"Backup Master for your website" plugin will allow to admin to create backup of database and files for wordpress store. This plugin provides backup of files and database backups with a single click.


== Description ==
* Author: [Suncart Store](https://www.suncartstore.com)

"Backup Master for your website" is the greater tool to create backup of database and files for your wordpress store. With a single click, admin can create Backup files and a database

====Backup Master for your website Features====
> - Single click database and files backup.
> - Easily downloading and deletion of generated backups.
> - Upto 10 recent backups.

You never want to loose your store, right?

A web store is not just a website, it is your dream and entire investment. As protection is better than cure, your store website can get down in case of hacking attempts, wordpress update goes fail, hosting expired, server crashing and you may lose everything. Why don't we take an extra step to prepare a backup before anything negative occurs?

== Installation ==
= Backup Master for your website =
1. Go to Plugins > Add New Plugin*, search for **Backup Master for your website** and click "Install now"
2. Alternative method, download the plugin and Upload the Backup Master for your website folder to the /wp-content/plugins/directory
3. Activate the plugin
4. Now Go to Settings > Backup Master for your website. Choose the option as database only or both database & files.
5. Generate backup. Within snap of fingers your backup will be generated.
6. That's all

OR, See detailed doc, [how to install a WordPress plugin](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

= Is this plugin capable to backup the uploads and plugins also? =
Yes, for that you need to select the option "Both database & files" and click on generate backup.
